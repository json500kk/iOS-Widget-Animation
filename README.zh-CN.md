# iOS 27 小组件动画

面向主屏幕小组件的商业化 iOS 动画 SDK，完美支持 **iOS 27**，解决 **ClockHandRotationKit** 在 iOS 27 上无法正常工作的问题，同时兼容 **ClockHandRotationKit** 的迁移需求，并支持 TopWidget 风格的 **SwingAnimation** 动画效果。

## 概述

`iOS Widget Animation` 是一个闭源 SDK，用于在 iOS 上构建流畅、可用于生产环境的主屏幕小组件动画。

本公开仓库仅用于效果展示，不包含私有源码实现。

## 亮点

- 支持 iOS 27 主屏幕小组件动画
- 兼容 ClockHandRotationKit，便于轻松迁移
- 兼容 TopWidget 风格的 SwingAnimation 动画效果
- 专为流畅的时间驱动动画而设计
- 提供简洁公开 API，支持商业集成

## 兼容性

### iOS 27 小组件动画

该 SDK 专为支持 iOS 27 主屏幕小组件的动画行为而构建，重点在于视觉过渡的流畅性。

### ClockHandRotationKit 迁移兼容

如果你的项目已经在使用 `ClockHandRotationKit`，该 SDK 可以在保留熟悉概念的同时降低迁移和接入成本。

### TopWidget SwingAnimation 兼容

如果你希望实现与 `TopWidget` 的 `SwingAnimation` 类似的动画效果，该 SDK 提供兼容这一风格的运动能力支持。
