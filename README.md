# iOS 27 Widget Animation

Commercial iOS widget animation SDK for Home Screen widgets, with perfectly supports iOS 27 **iOS 27** support, **ClockHandRotationKit** migration compatibility, and TopWidget **SwingAnimation-style** effects.

## Overview

`iOS Widget Animation` is a closed-source SDK for building smooth, production-ready animated Home Screen widgets on iOS.

This public repository is a showcase only and does not include the private source code.

## Highlights

- Supports animated Home Screen widgets on iOS 27
- Compatible with ClockHandRotationKit for easier migration
- Compatible with TopWidget-style SwingAnimation effects
- Designed for smooth time-based animation
- Commercial integration available with a clean public API

## Compatibility

### iOS 27 Widget Animation

The SDK is built to support animation behaviors for iOS 27 Home Screen widgets, with a focus on visually smooth transitions.

### ClockHandRotationKit Migration

If your project already uses `ClockHandRotationKit`, this SDK is designed to make migration easier by preserving familiar concepts and reducing integration friction.

### TopWidget SwingAnimation Compatibility

If you want animation behavior similar to `TopWidget`'s `SwingAnimation`, this SDK provides compatible motion capabilities for that style of effect.
