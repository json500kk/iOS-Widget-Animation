# iOS 27 小工具動畫

面向主畫面小工具的商業化 iOS 動畫 SDK，完整支援 **iOS 27**，解決 **ClockHandRotationKit** 在 iOS 27 上無法正常運作的問題，同時相容 **ClockHandRotationKit** 的遷移需求，並支援 TopWidget 風格的 **SwingAnimation** 動畫效果。

## 概述

`iOS Widget Animation` 是一套閉源 SDK，用於在 iOS 上打造流暢、可用於正式環境的主畫面小工具動畫。

此公開儲存庫僅用於效果展示，不包含私有原始碼實作。

## 亮點

- 支援 iOS 27 主畫面小工具動畫
- 相容 ClockHandRotationKit，便於輕鬆遷移
- 相容 TopWidget 風格的 SwingAnimation 動畫效果
- 專為流暢的時間驅動動畫設計
- 提供簡潔公開 API，支援商業整合

## 相容性

### iOS 27 小工具動畫

此 SDK 專為支援 iOS 27 主畫面小工具的動畫行為而設計，重點在於視覺轉場的流暢度。

### ClockHandRotationKit 遷移相容

如果你的專案已經使用 `ClockHandRotationKit`，此 SDK 能在保留熟悉概念的同時，降低遷移與整合成本。

### TopWidget SwingAnimation 相容

如果你希望實現與 `TopWidget` 的 `SwingAnimation` 類似的動畫效果，此 SDK 提供相容該風格的動態能力。
