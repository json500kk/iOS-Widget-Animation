# iOS 27 위젯 애니메이션

홈 화면 위젯을 위한 상용 iOS 애니메이션 SDK로, **iOS 27**을 완벽하게 지원하고 iOS 27에서 **ClockHandRotationKit** 이 정상적으로 동작하지 않는 문제를 해결합니다. 또한 **ClockHandRotationKit** 마이그레이션 호환성과 TopWidget 스타일의 **SwingAnimation** 효과도 지원합니다.

## 개요

`iOS Widget Animation`은 iOS 홈 화면 위젯에서 부드럽고 실제 서비스에 적용 가능한 애니메이션을 구현하기 위한 클로즈드 소스 SDK입니다.

이 공개 저장소는 데모 및 소개용이며, 비공개 소스 코드는 포함되어 있지 않습니다.

## 주요 특징

- iOS 27 홈 화면 위젯 애니메이션 지원
- ClockHandRotationKit과 호환되어 손쉬운 마이그레이션 가능
- TopWidget 스타일의 SwingAnimation 효과 지원
- 부드러운 시간 기반 애니메이션을 위해 설계됨
- 깔끔한 공개 API를 통한 상용 통합 지원

## 호환성

### iOS 27 위젯 애니메이션

이 SDK는 iOS 27 홈 화면 위젯의 애니메이션 동작을 지원하도록 설계되었으며, 시각적으로 부드러운 전환에 중점을 둡니다.

### ClockHandRotationKit 마이그레이션 호환

기존 프로젝트에서 `ClockHandRotationKit`을 사용하고 있다면, 이 SDK는 익숙한 개념을 유지하면서 마이그레이션과 통합 부담을 줄일 수 있도록 설계되었습니다.

### TopWidget SwingAnimation 호환

`TopWidget`의 `SwingAnimation`과 유사한 애니메이션 동작이 필요하다면, 이 SDK는 해당 스타일에 맞는 모션 기능을 제공합니다.
